# Zahoor Personal Website

This project is my personal website and future company-facing website, built with Next.js and deployed on Vercel.

## Current status

The project started as a simple Hello World site to validate the local-to-GitHub-to-Vercel workflow.

That workflow is now working:
- local development works
- GitHub integration works
- Vercel deployment works
- production site is live on a Vercel subdomain
- later this site will be connected to `zahoorahmed.com`

## Main goal

Build a modern, fast, credible personal brand website that can evolve from a simple portfolio into a more advanced website with dynamic sections, lead capture, product pages, and possible app integration.

This is not meant to stay a basic portfolio forever. It should be built in a way that supports future growth.

## Design direction

The design should be:
- modern
- clean
- minimal but premium
- mobile-first
- responsive
- fast-loading
- readable
- visually credible for a technical founder / builder

Avoid:
- clutter
- too many colors
- gimmicky animations
- template-looking sections with no real purpose
- overly generic "agency" vibe

Preferred visual style:
- strong typography
- generous spacing
- clean section hierarchy
- subtle modern polish
- simple but high-quality UI
- dark mode can be considered later, but not required initially
